Solution Overview & Benefits

IndiGo is a complete, low cost solar energy system for homes without access to municipal power. The system includes a solar panel, control box and LED lighting. Batteries can be attached to the control box to store electricity for nighttime use. As well as lights, control box can charge cell phones and power other small devices.

The system is paid for by the user as they need it. Cards are available for purchase that contain a code. The code is sent by text message to Azuri Technologies which responds with a number. The number is then entered into the control box and the system can be used for the amount of time purchased. One week of use currently costs approximately $1. Azuri Technologies hopes this payment method will replace paying for the system upfront which is prohibitively expensive for some customers.

History & Development

IndiGo was developed by Eight19, a UK firm which primarily develops solar cells. Eight19 launched Azuri Technologies to carry Indigo pay-as-you-go solar in 2012.
